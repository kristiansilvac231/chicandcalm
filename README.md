# Chic & Calm

Landing page para **Chic & Calm**, asesoría de imagen personal y corporativa en Chile, a cargo de Camila Díaz — Relacionadora Pública con más de 20 años de experiencia.

## Servicios

- Asesoría de imagen personal
- Colorimetría
- Imagen ejecutiva y marca personal
- Personal shopper
- Capacitación corporativa

## Stack

Sitio estático HTML/CSS/JS — sin dependencias ni build step.

## Deploy

Desplegado en [Vercel](https://vercel.com). Cualquier push a `main` publica automáticamente.

```bash
# Vista previa local
open index.html
```
